# 中国罕见病照护者喘息资源索引

## 1. 中国喘息服务概览

**现状**：国内喘息服务（respite care）体系尚处起步期。
- **政策层面**：2016 年《"健康中国 2030"规划纲要》首提长期护理；2016 年起
  上海、青岛等 15 个城市试点长期护理保险（"长护险"）；2020 年扩至 49 城。
- **落地层面**：试点集中在北上广深杭苏、宁波、成都、重庆等一线/强二线。
  多数中西部城市仍无成体系的家庭喘息服务。
- **罕见病特殊性**：罕见病多为儿童期发病、长期甚至终身照护，但大多未被
  纳入"长护险"（多数长护险只覆盖失能老人）。罕见病家庭往往依赖自费家政、
  家庭分担与公益组织短期支持。

**重要提醒**：本文信息会随政策变化更新。具体申请条件/额度以当地民政局、
卫健委、残联、医保局的**最新公开公告**为准。任何索要"代办费""加速费"
的个人或机构几乎 100% 是骗局，请直接向政府窗口咨询。

## 2. 各类资源（按层级从近到远）

### 2.1 家庭内分担

最直接、最可持续的喘息来源。核心是把"我一个人扛"变成"我们一起分"。

**时间表模板**（以罕见病儿童为例）：
```
            周一   周二   周三   周四   周五   周六   周日
6-8 早护理  主照   配偶   主照   配偶   主照   爷奶   主照
上午医疗    主照   主照   —      主照   —      —      —
下午陪伴    老人   老人   主照   老人   配偶   配偶   主照
晚 21-23    配偶   主照   配偶   主照   配偶   主照   配偶
```

**沟通脚本**（请求家人分担时）：
- "我需要每周有半天完全不管 XX 的事，否则我撑不了多久。你能接周六下午吗？"
- "医疗的事我来管，但洗澡/喂饭我希望你能固定接 3 次/周。"
- 避免："你从来都不帮忙"（指责） → 改为："这三件事我需要你固定接手。"

### 2.2 社区资源

| 资源 | 承接机构 | 适用 |
|---|---|---|
| 居家护理上门 | 社区卫生服务中心 / 家庭医生签约 | 换药、导尿、鼻饲指导、压疮护理 |
| 日间照料中心 | 街道/社区（部分含失能儿童） | 半日/全日托管，多为老年失能，罕见病儿童接收有限 |
| 辅具租借/发放 | 区县残联辅具服务中心 | 轮椅、护理床、呼吸机等 |
| 康复训练补贴 | 残联"残疾儿童康复救助" | 0-14 岁（部分地区 0-17）持证残疾儿童每年数千至数万元康训费 |
| 临时看护 | 社区志愿者团队、民政购买服务 | 北上广深部分街道试点 |

建议直接电话咨询**户籍所在地街道办民政科**与**区县残联**，报"家里有重病/
罕见病患者想了解居家照护服务有哪些"，让对方列出本街道可用清单。

### 2.3 公益组织喘息项目

以下仅列出目前有公开运营记录的全国性组织，具体项目请以其官网/公众号
最新公告为准：

- **瓷娃娃罕见病关爱中心**（北京）：成骨不全症社群起家，覆盖多个罕见病
  群体，有家长互助营、照护技能培训、紧急救助等。
- **蔻德罕见病中心（CORD）**：研究+倡导为主，部分患者支持项目，
  组织罕见病家庭大会（每年）。
- **病痛挑战基金会**：罕见病医疗援助信息平台+部分援助金。
- **中国罕见病联盟**：官方背景，整合三甲医院罕见病诊疗协作网信息。
- **血友之家**：血友病群体照护互助。
- **月亮孩子之家**：白化病群体。
- **中国 SMA 之家**：脊髓性肌萎缩症。
- **美儿 SMA 关爱中心**：SMA 患者家庭支持。

**用法**：
1. 病种对口 → 先找病种专属组织，其次找瓷娃娃/CORD 等综合组织。
2. 公益喘息项目通常**不是单纯"替你看孩子"**，更常见的是：家长培训营、
   夏令营期间托管、照护者心理支持小组、同伴互助微信群。
3. 加入前先关注官方公众号、确认机构合法注册（民政部"中国社会组织政务
   服务平台"可查 → https://www.chinanpo.mca.gov.cn/ ）。

### 2.4 医院专业喘息

- **儿童医院**：部分三甲儿童医院设有长期住院患儿的家长休息室、志愿者
  陪护班（多为公益机构对接）。短期住院托管极少，多数需家长全程陪护。
- **安宁疗护/缓和医疗病房**：针对终末期患者，家属可短期"放下"，专业
  团队接管。试点医院见国家卫健委三批安宁疗护试点名单（北京海淀、
  上海、长春、德阳、洛阳等）。
- **康复医院 / 护理院**：非公立康复/护理院有短期托管服务（1-4 周），
  自费为主，每月数千至上万元。

### 2.5 线上互助资源

- **病友/家属微信群**：通过瓷娃娃、CORD、病种专属组织报名后入群。
- **罕见病妈妈圈**：部分组织运营的家长互助社群。
- **互助问答平台**：丁香医生罕见病板块、知乎相关话题、小红书 #罕见病
  照护# 等（信息质量参差，作参考不作决策依据）。
- **远程心理支持**：简单心理、KnowYourself、壹心理等有公益计划覆盖
  部分困难家庭；武志红心理线上小组（付费）。

## 3. 政府补贴与政策

### 3.1 困难家庭临时救助

- **适用**：因病支出超过家庭承受能力。
- **申请地点**：户籍所在地**街道办民政科**。
- **额度**：视地方与困难程度，一般数千至数万元一次性救助。
- **材料**：低保证/困难证明、医疗支出凭证、户口本、身份证。

### 3.2 残疾人两项补贴（全国政策）

两项独立补贴，可叠加领取：

| 补贴 | 对象 | 额度参考 |
|---|---|---|
| **困难残疾人生活补贴** | 低保户中的持证残疾人 | 50-200 元/月（各地不同） |
| **重度残疾人护理补贴** | 持证**一级或二级**残疾人 | 50-300 元/月（各地不同） |

- **前提**：需先办"残疾人证"（户籍地区县残联申请，含医学鉴定）。
- **罕见病患者**：能否办证取决于残疾类别和程度。肢体/视力/听力/言语/
  智力/精神/多重——按这七类鉴定，部分罕见病需经医学鉴定符合等级才能办。

### 3.3 儿童大病救助

- **国家医保特殊补助**：部分罕见病用药已纳入医保目录（SMA、戈谢病、
  庞贝病等药物近年陆续进入）；地方医保会有大病专项报销。
- **大病医保（城乡居民）**：超出封顶后分段报销。
- **儿童白血病、先心病专项救助**：民政部+红十字会专项。
- **中国红十字基金会**：小天使基金（白血病）、天使阳光（先心病）等。
- **中华慈善总会**：数十种罕见病药品援助项目，详情见 crcf.org.cn 与
  官方公众号。

### 3.4 地方特色政策

- **上海**：长护险覆盖失能老人居家/机构照护（罕见病儿童不在主范围，但
  若伴随严重失能可争取评估）；部分区试点"喘息服务"（家属短期脱离照护）。
- **北京**：长护险在石景山试点；部分街道有针对失能重残的临时托管补贴。
- **杭州、苏州、青岛**：长护险覆盖较广，待遇有差异。
- **广州、深圳**：深圳有"重残儿童康复训练救助"等单列项目。
- **成都、重庆**：长护险试点中；成都有部分医院开展罕见病门诊绿色通道。

**查询方法**：
- 搜索"〔城市名〕+长期护理保险+试点"
- 搜索"〔城市名〕+残疾人两项补贴+标准"
- 拨打政府 **12345 热线**询问本地具体政策与申请窗口。

## 4. 申请流程模板（以"残疾人两项补贴"为例）

### 所需材料

1. 申请人身份证（复印件 + 原件）
2. 户口本
3. 残疾人证（一级/二级用于护理补贴）
4. 低保证或困难家庭证明（用于生活补贴）
5. 银行卡（发放账户，通常需本人或监护人名下）
6. 近期证件照 1-2 张
7. 填写申请表（当地民政窗口领取）

### 申请地点

- 户籍所在地**街道办事处（乡镇民政办）**的民政窗口。
- 部分地区支持"一网通办"线上申请（当地政务服务网）。

### 审核周期

- 街道初审 10 个工作日 → 区县民政复核 10 个工作日 → 公示 → 发放。
- 从提交到首次发放通常 **1-3 个月**。

### 发放方式

- 按月或按季打入申请人（或监护人）指定银行账户。
- 审核通过后从申请次月开始计发。
- 每年年检一次（部分地方年审在线完成）。

### 常见拒绝原因

- 残疾等级不够（如三级/四级不符合护理补贴）→ 可申请残疾等级复核。
- 家庭收入超低保线 → 生活补贴不批，但护理补贴只要等级够仍可批。
- 材料不全 → 按窗口回执补齐再提交。

## 5. 经济支持外的实用资源

### 5.1 托育 / 上学衔接

- **特殊教育学校**：户籍地教育局可查学区对应特校清单。
- **普通学校随班就读**：《残疾人教育条例》保障零拒绝入学；若遇拒收可
  向区县教育局投诉。
- **送教上门**：重度不能到校者由辖区学校送教，每周 1-2 次。
- **医教结合**：部分三甲医院 + 特校合作项目（如上海、北京、杭州）。

### 5.2 辅助技术与设备

- **轮椅/护理床/呼吸机等租赁**：区县残联辅具服务中心（免费或低价配发）。
- **康复机器人/外骨骼**：三甲康复科评估使用。
- **AAC 沟通辅助**：面向无法言语者的图片/平板沟通系统，相关基金会项目
  可申请（如病痛挑战基金会、部分地方康复中心）。

### 5.3 法律援助

- **监护权 / 意定监护**：父母预先指定"如果我们不在"谁来照顾患儿——
  参考《民法典》第 33 条成年意定监护、第 27 条未成年人指定监护。
  可到公证处办理。
- **遗嘱 + 特殊需求信托**：为残障家庭成员设立信托资产（国内仍发展中，
  中信、建信、光大等部分大型信托公司提供家族信托含特殊需求条款）。
- **免费法律援助**：司法局"12348"法律援助热线；各地法律援助中心。
- **残疾人法律维权**：中国残联法律维权部（010-65731299 等）与各省残联。

## 6. 声明与安全提示

- 资源信息**动态变化**。本文所列机构与政策以发文时为准，申请前务必核对
  官方最新公告。
- 所有政府补贴**不收代办费**。任何要求先支付"申请费""加急费""打点费"
  的几乎都是诈骗，可直接拨打 12345 或报警 110 举报。
- 公益组织申请原则上也**不收费**；若项目涉及成本分摊（如夏令营食宿），
  应有明确公示的票据与财务公开。
- 个人隐私保护：申请各类援助时，注意诊断报告、医保卡号、银行账户信息
  只在官方渠道/正规机构之间传递。
- 遇到无法判断真伪的项目，先到"中国社会组织政务服务平台"查该组织是否
  注册，再到公开渠道搜索"〔组织名〕+骗局"等负面关键词做尽调。
