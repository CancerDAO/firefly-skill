# 遗传风险家庭告知 · 伦理与脚本参考

> 操作手册。不是伦理学论文。给家庭成员用"直接能抄的话"。
> 配套 `firefly-disclosure/SKILL.md`。

---

## 1. 四类告知场景定义

| 场景 | 告知对象 | 伦理核心 | 关键张力 | 典型困境 |
|---|---|---|---|---|
| **(A)** | 未成年患者本人 | 知情权发展中 (evolving autonomy) | 保护 vs 诚实 | 3 岁孩子反复问"我为什么要打针"；13 岁青春期自己上网搜出 Orphanet 页面 |
| **(B)** | 成年血亲（兄弟姐妹/父母/成年子女） | 告知义务 (duty to warn) | 先证者隐私 vs 亲属健康/生育权 | 姐姐备孕中；哥哥拒绝做检测；父母情绪崩溃自责 |
| **(C)** | 配偶 | 生育共同决策 + 婚内诚信 | 携带者自我污名 vs 伴侣知情权 | 孕 16 周发现胎儿携带者；WES 副发现揭示非亲子 |
| **(D)** | 延展家族（堂表亲/远亲） | 群体健康 + 尊重距离 | 关系疏远 vs 医学风险告知 | 表妹只在年夜饭见过；二叔已去世但其子女有风险 |

## 2. 伦理原则速查

| 原则 | 本场景含义 | 冲突点 |
|---|---|---|
| 知情权 (right to know) | 亲属有权了解影响自己健康/生育的遗传信息 | 与先证者保密权冲突 |
| 自主权 (autonomy) | 被告知者有权决定是否接受检测 / 生育选择 | 与家族"都该查"的压力冲突 |
| 不伤害 (non-maleficence) | 告知可能造成污名、家庭破裂、失业/保险歧视 | 与告知的医学收益权衡 |
| 公正 (justice) | 所有处于风险的亲属都应被平等告知 | 实务中常只告知"亲近的" |
| 保密义务 vs 告知义务 | 医生对先证者保密，但亲属面临可预防重大伤害时例外 | ACMG/ASHG：先证者应为首选告知者 |

**核心立场引用**：

- **ACMG SF v3.1 (2021)**：应对 81 个基因的次要发现 (secondary findings) 主动分析并告知，未成年人亦包括成人发病疾病（BRCA1/2、LS、HCM 相关基因）。家长可 opt-out。
- **NSGC Position on Duty to Warn (2018)**：咨询师支持先证者自主告知亲属；极端情况（严重可预防伤害 + 先证者拒绝 + 可识别亲属）才考虑直接告知，须经伦理审查。
- **中华医学会医学遗传学分会《遗传咨询专家共识 (2019)》**："先证者主动告知"为首选路径，咨询师提供"家族信"模板。
- **《中国人类遗传资源管理条例 (2019)》**：跨境共享基因数据受约束；家庭内部告知不受限。
- **AAP Adolescent Consent (Committee on Bioethics, 2016)**：12+ 青少年应参与自身医疗决策；14+ 在预测性检测中的 assent 权重显著上升。

## 3. 按年龄分层告知脚本（场景 A）

参考 SHARE 模型 / Wiener 等 (2012) 儿科披露框架。

### 3.1 0-6 岁

**原则**：不主动告知病名。不撒谎。用感官语言回答具体问题。

- **开场**（孩子问"我为什么要去医院"）："宝贝，你身体里有个小小的地方需要医生帮它修一修，就像玩具坏了要去修理店。"
- **中段**："医生会听心跳、抽一点点血。像小蚂蚁咬一下，咬完就过去了。妈妈全程牵着你的手。你可以哭，可以喊。"
- **结尾 + 资源**："回家吃你爱的草莓酸奶。" → Child Life Specialist（上海儿医/北京儿研所/深圳儿童）、绘本《当我生病的时候》。

### 3.2 7-11 岁

**原则**：部分披露。用故事/隐喻。允许情绪。不一次讲完整遗传学。

- **开场**："妈妈想和你说一件重要的事，关于你身体。我们找个安静的时间好好聊，不着急。"
- **中段**："你身体里有一本说明书，告诉每个细胞怎么工作。每个人的说明书都有小错字。你的说明书里有一个错字，让你的 [肌肉/肾/血液] 比别人需要更多照顾。这不是你做错了什么，不是谁传染的，是你出生就带着的。医生知道怎么帮你。你可以问任何问题，我不知道的我们一起查。"
- **结尾 + 资源**："你现在是什么感觉？难过、害怕、没感觉都可以说。" → 儿童遗传咨询师陪同二次谈话、病痛挑战基金会/蔻德 CORD 儿童活动。

### 3.3 12-17 岁

**原则**：全披露。病名 + 机制 + 预后范围 + 生育未来。AAP 2016：12+ 须获得其 assent。

- **开场**："我想把关于你这个病的完整信息告诉你。你已经长大了，这是你的身体、你的未来，你有权知道全部。可以随时打断我。"
- **中段**："你的诊断叫 [病名，中英文都给]。原因是 [基因名] 上一个变异，是 [显性/隐性/X 连锁]，意味着 [对后代风险]。现在治疗能做到 [X]，还做不到 [Y]。过去 5 年出了 [具体药/试验]。以后生育有 PGT（胚胎植入前检测）这样的选项，我们可以慢慢讨论。这些一次消化不完，我打印了一份给你。"
- **结尾 + 资源**："你想自己先读再聊，还是现在就问？" → 同龄病友社群、壹心理青少年专线、遗传咨询师一对一（可不带家长）。

### 3.4 成年

**原则**：完整披露 + 生育咨询 + 亲属告知讨论。

- **开场**："有些信息以前没有完整告诉你，因为你还小。现在你成年了，应该全部知道。"
- **中段**：完整基因型/表型/预后/生育风险/亲属风险/保险歧视风险。
- **结尾**：预约成人遗传咨询门诊；讨论"谁来告诉其他亲属"。

## 4. 成年血亲告知脚本（场景 B）

### 4.1 打开话题的 3 种方式

1. **直接式**（关系亲近、对方情绪稳定）：
   > "姐，我这次做的基因检测出了一个结果，可能对你也有影响。我想亲口告诉你。你现在方便听吗？"
2. **过渡式**（关系一般或对方正忙）：
   > "哥，我最近做了一些医学检查。有个结果医生说家里兄弟姐妹最好也了解一下。什么时候你有半小时，我们视频聊一下？"
3. **信件/家族信式**（远亲 / 表达口头困难 / 关系紧张）：见 §6.2 模板。

### 4.2 信息层级（从"需要知道"→"有风险"→"可以做什么"）

```
L1 你需要知道的：
   我被确诊 [病名 / 携带 XX 基因变异]。这是遗传性的。
L2 对你意味着：
   作为我的 [关系]，你有 [50% / 25% / 可能] 的概率也携带这个变异。
   如果携带，对你的健康意味着 [X]，对你孩子的风险是 [Y]。
L3 你可以做什么：
   a. 不做任何事（这也是一个合法选项）
   b. 找遗传咨询师讨论，再决定要不要检测
   c. 直接做检测（单基因测序，约 XXX 元，XX 天出结果）
   我可以把我的医生/咨询师介绍给你。
```

### 4.3 按外显率调整

| 变异类型 | 告知紧迫度 | 措辞调整 |
|---|---|---|
| **高外显致死/致残**（HD, BRCA1, LS, HCM） | 高 | 明确"强烈建议考虑检测"；提供直接转诊路径 |
| **低外显 / 条件性** | 中 | 强调"信息价值"而非"行动必要"；留更多自主空间 |
| **VUS (意义未明)** | 低 | 必须说明"目前尚不确定有害"；避免造成无谓焦虑；建议家族共分离研究 (co-segregation) |

## 5. 配偶告知脚本（场景 C）

### 5.1 备孕前发现携带者

- **开场**："我做了孕前筛查，发现自己是 [病名] 携带者。我本人不会发病，但如果你也是携带者，孩子有 25% 概率得病。我想和你一起决定下一步。"
- **后续菜单**：伴侣检测 → 若双方携带：自然妊娠 + 产前诊断 / PGT-M / 供精供卵 / 领养 / 不生育。
- **关键**：不替伴侣决定"要不要做检测"。

### 5.2 已孕发现

- **开场**："刚才医生打电话告诉我产检结果。有个我们需要一起面对的消息。你现在坐着方便吗？想当面告诉你。"
- **信息顺序**：事实 → 医学选项（继续妊娠 / 治疗性引产 / 进一步确诊）→ 时间窗 → 情绪空间。
- **资源**：产前诊断中心遗传咨询、终止妊娠法律咨询（中国 28 周前因严重遗传病可依法选择）。

### 5.3 非亲子/非亲缘发现（misattributed parentage）

**伦理红线**：非请求性披露此类信息属高风险操作。

- **先与遗传咨询师/伦理委员会复核**：是否必须告知？是否直接影响医学决策？
- **默认不主动披露给配偶之外的第三方**（不告诉孩子、不告诉其他亲属）。
- **分别告知协议**：先与生物学母亲单独沟通 → 她决定是否/如何告知配偶 → 咨询师提供在场支持。
- **脚本（咨询师对母亲）**："检测结果里有一项发现，和医学诊断没有直接关系，但可能对你个人有意义。这类信息你希望我告诉你吗？"（给 opt-out 权）
- 参考：ASHG Statement on Incidental Findings (2013)、ACMG SF v3.1。

## 6. 延展家族告知（场景 D）

### 6.1 间接 vs 直接

- **默认间接**：通过最近的共同亲属转达（先证者 → 父母 → 叔伯 → 堂表亲）。
- **直接告知**的触发条件：高外显 + 可预防 + 近亲已故或失联。

### 6.2 家族信模板

**中文版**：

```
亲爱的 [家族成员姓名]：

见信如面。写这封信是因为有一个健康信息，我觉得有义务告诉你。

最近我（或我的 [孩子/配偶]）被确诊 [病名 / 英文名]。医生经过基因检测，
确认这是由 [基因名] 基因上的一个遗传变异引起的。这种变异可能在家族中
传递。

作为我的 [具体关系，如"堂妹"]，你有大约 [X]% 的概率也携带这个变异。
携带不等于发病，但了解自己的状态可以帮助你：
  1. 对自己的健康做出更知情的安排；
  2. 如果你在考虑生育，可以提前做风险评估。

如果你希望进一步了解，以下资源可能有用：
  - Orphanet 页面：https://www.orpha.net/ORPHA{编号}
  - 我的遗传咨询师：[姓名、医院、联系方式，已取得对方同意]
  - 国内相关患者组织：[蔻德罕见病中心 / 病痛挑战基金会 / 病种专属组织]

你不需要立刻回复，也不需要做任何决定。我尊重你处理这个信息的方式和
节奏。如果你想聊，随时可以联系我。

[先证者姓名]
[日期]
```

**English version**（跨国家庭）：

```
Dear [Name],

I'm writing to share health information you should know. I (or my
[child/spouse]) was recently diagnosed with [condition]; genetic
testing confirmed a pathogenic variant in [GENE], which can be passed
down through our family.

As my [relation], you have approximately [X]% chance of carrying this
variant. Being a carrier does not mean you will get sick, but knowing
your status helps you make informed decisions about health and,
if relevant, family planning.

Resources: Orphanet (https://www.orpha.net/ORPHA{number}), GeneReviews
(NCBI), my genetic counselor [name, with permission]. No urgency to
respond or act — I respect whatever pace works for you.

[Name] / [Date]
```

## 7. 家庭会议协议

| 维度 | 建议 | 避免 |
|---|---|---|
| **时机** | 非节假日、非生日、非重大家庭事件（高考/婚礼后）；周末下午；所有参与者次日无重大安排 | 年夜饭、春节、他人生日、重要考试/面试前 |
| **环境** | 家中私密空间 / 咨询室；手机静音但不关机（急事可接）；纸巾、温水、零食备好 | 餐厅、咖啡馆、有外人进出的客厅 |
| **时长** | 至少预留 90 分钟；不定硬性结束时间 | "15 分钟说完"、工作间隙 |
| **参与者** | **第一轮**：先证者 + 最核心家属（配偶/父母）<br>**第二轮**：扩展核心（成年兄弟姐妹、成年子女）<br>**第三轮**：延展（叔伯堂表，可用信件） | 一次把 10+ 人叫在一起 |
| **陪伴者** | 遗传咨询师（首选，能回答技术问题）/ 家庭医生 / 家族信任的年长者 / 心理咨询师 | 不熟悉的旁观者、儿童（除非本场景就是对儿童告知） |
| **准备材料** | 一页 A4 诊断摘要（病名、基因、遗传模式、外显率、家族风险表）、Orphanet 或 GeneReviews 打印版 | PPT、长报告（压迫感） |

## 8. 拒绝/情绪反应的 4 种预案

### 8.1 否认 ("不可能是我家")

- **反映**："我听到了，这听起来完全不像我们家会有的事。"
- **不辩解，给资料**："这是检测报告和医生联系方式。你不用现在接受，放着就好。"
- **时间**："过几天/几周你想再聊，我随时都在。"

### 8.2 愤怒 ("你早就知道了"/"你凭什么说我有病")

- **接纳**："你生气是合理的，我理解。"
- **不辩解，陈述事实**："我知道这个结果是 [日期]，报告在这里。我想过各种告诉你的方式，最后选了现在。"
- **不反击**：允许对方把气撒完；不当场要求原谅。

### 8.3 悲伤

- **共同静默**：不填补沉默，不说"别哭了"。
- **允许哭泣**：递纸巾，可握手/拥抱（视关系）。
- **陪伴脚本**："我在这里。不用说话。"
- **后续**：情绪过了再说下一步，不在哭的当下安排行动。

### 8.4 接受

- **不被感动冲昏头脑，落地行动**："谢谢你愿意听。我们列一下接下来 2 周各自可以做的事？"
- **清单**：预约遗传咨询 / 决定要不要检测 / 告诉谁不告诉谁 / 下次复盘时间。

## 9. 特殊情境

### 9.1 孩子成人前 vs 成人后分享

- **成人前**（12-17）：按 §3.3，以孩子的医疗需要为锚，父母为主导。
- **成人后**（18+）：告知权交还本人；父母从"决定者"转为"支持者"。
- **断档风险**：最糟糕的是孩子 17 岁前完全蒙在鼓里，18 岁生日那天被一次性全盘告知——应避免，分层过渡。

### 9.2 未成年人次要发现 (secondary findings)

- **ACMG SF v3.1 (2021)**：临床外显子/基因组测序中，对 81 个基因的致病变异应主动分析并报告，未成年人亦不例外。但：
  - 家长有 **opt-out** 权（签署知情同意时）；
  - 成人发病疾病（如 BRCA1/2）虽报告给家长，建议告知孩子的时机推迟到其 **成年可独立决策** 时；
  - 儿科遗传咨询师应帮助家长规划告知时间表。
- **常见纠结**：
  - "孩子 10 岁查出 BRCA1，要告诉他吗？" → 建议：现在不主动告知孩子，18 岁时分层披露；但家长自己 **应立即** 做级联检测 (cascade testing)。

### 9.3 跨国家庭

- **语言**：准备中英双语家族信、Orphanet 页面（Orphanet 本身多语）、GeneReviews（英文权威）。
- **医疗系统差异**：美国/欧洲亲属可用 NSGC/ESHG 当地咨询师；提供 FindAGeneticCounselor.com 链接。
- **保险歧视**：美国有 GINA (2008) 保护健康保险+就业不受基因歧视（但不覆盖寿险/长护险）；中国无对应专门立法，告知时提醒亲属谨慎对外透露（单位体检、商业保险购买）。
- **跨境基因数据**：按《中国人类遗传资源管理条例 (2019)》，向境外提供原始基因数据需审批；家庭内部口头/书面告知（不含原始数据）不受限。
