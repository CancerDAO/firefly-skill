# PGT-M 胚胎植入前单基因病遗传学检测参考

## 1. PGT-M 概念

**Preimplantation Genetic Testing for Monogenic disorders**（前称 PGD - Preimplantation Genetic Diagnosis），即胚胎植入前单基因病遗传学检测。通过对体外受精 (IVF) 所得胚胎进行活检和分子检测，选择不携带家族致病变异的胚胎移植，从源头阻断单基因遗传病传递。

### PGT 三大类别
- **PGT-M**：单基因病（Monogenic），本文件重点
- **PGT-A**：非整倍体筛查（Aneuploidy），针对高龄、RPL 等
- **PGT-SR**：结构重排（Structural Rearrangement），针对染色体易位、倒位携带者

## 2. 适应证（中国法规约束）

依据国家卫健委《人类辅助生殖技术规范》《胚胎植入前遗传学诊断技术规范》，PGT-M 限定：

- 夫妻双方或一方为**已知致病变异携带者**（P 或 LP 级，不接受 VUS）
- 该遗传病对子代有**严重健康影响**（严重致残、致畸、致死或已明确重度临床表现）
- 须通过所在医院**医学伦理委员会审查**
- 不允许为非医学目的（性别选择、容貌、能力等）
- 产前诊断（CVS、羊水穿刺）仍为法定补充方案，夫妇可选择常规妊娠 + 产前诊断

### 典型适应证疾病
- 高外显重度 AD：HD、NF1 恶性表型、遗传性结直肠癌 (FAP)、遗传性乳腺癌 (BRCA1/2 家族请求)
- AR：SMA、DMD（XLR）、β-地中海贫血、苯丙酮尿症、Tay-Sachs
- X 连锁：DMD/BMD、血友病、Fragile X
- 染色体结构异常：平衡易位、罗伯逊易位
- 线粒体病（PGT-M 价值有限，异质性难预测）

- 周期等待时间（部分顶级中心排队 6-12 个月）
- 伦理审批效率

## 4. 流程与时间线

```
遗传咨询 → 确诊变异 → 家系扩展（informative marker 建立） → 伦理审批
   → 促排卵 (Gn 8-14 天)
   → 取卵（通常 10-20 枚卵母细胞）
   → IVF 或 ICSI 体外受精
   → 胚胎培养至 Day 5/6 囊胚
   → TE (滋养外胚层) 活检（5-8 细胞）
   → WGA 全基因组扩增 + PGT-M 分子检测（1-4 周）
   → 胚胎冷冻（玻璃化 vitrification）
   → 等结果后，FET（冻胚移植）周期子宫内膜准备
   → 移植 → 妊娠监测 → 产前诊断确认（CVS 或羊穿）
```

**典型时间总览**：
- 术前咨询 + 家系建立 panel：1-3 个月
- 一个 IVF + PGT 周期：2-3 个月（含促排、取卵、检测、等待）
- 冻胚移植周期：1-2 个月
- **整体 3-6 个月**，如需多周期累加。复杂家系建立 panel 可能更长。

## 5. 费用（中国大陆，人民币）

| 项目 | 费用范围 | 备注 |
|------|---------|------|
| 促排卵药物 + IVF/ICSI 周期 | 3-8 万 | 因促排方案、药品种类浮动 |
| 胚胎培养 + 囊胚活检 | 1-2 万 | 每周期 |
| PGT-M 分子检测 | 2-5 万 | 含 panel 设计、WGA、测序、分析 |
| FET 冻胚移植周期 | 1-2 万 / 次 | 多次移植累加 |
| **单周期总计** | **~8-15 万** | 不含多周期与产前确认 |

医保覆盖情况（截至 2026 年）：
- **北京**：2023 年 7 月起将部分 IVF/ICSI 项目纳入医保（不含 PGT）
- **上海、广东（广州/深圳）**：陆续试点辅助生殖部分项目纳入医保
- **杭州、成都等**：地方补贴试点
- PGT-M 本身目前多数地区**不在医保报销范围**内，部分地区对罕见病家庭有专项补贴；建议咨询当地医保局

## 6. 成功率

### 6.1 年龄对 IVF 活产率的影响（每周期）
| 女方年龄 | 活产率（每取卵周期） |
|---------|------------------|
| <35 岁 | ~35-45% |
| 35-37 岁 | ~25-35% |
| 38-40 岁 | ~15-25% |
| 41-42 岁 | ~8-15% |
| >42 岁 | <10%（<5% 常见） |

### 6.2 PGT-M 检测准确率
- 单基因位点检测准确率 ~98-99%
- 结合单体型分析（SNP linkage）可降低 ADO (allele dropout) 风险
- 产前诊断仍建议在妊娠中做 CVS 或羊穿确认（residual misdiagnosis risk ~1-2%）

### 6.3 累积妊娠率
- 三周期累积活产率可达 60-70%（女方 <35 岁）
- 高龄患者可考虑多轮累积胚胎池策略

### 6.4 不孕不育叠加因素
如夫妇同时有不孕因素（输卵管、男方因素、卵巢功能下降），IVF 整体成功率会进一步下降。

## 7. 限制与风险

### 7.1 技术限制
- **胚胎嵌合 (Mosaicism)**：TE 活检结果可能不代表 ICM (内细胞团)。PGT-A 中已知嵌合胚胎移植可部分获得正常婴儿；PGT-M 中嵌合对单基因位点影响有限但需警惕
- **ADO (Allele Dropout)**：WGA 过程中某等位基因未扩增，可能误判；SNP 单体型辅助可降低但无法消除
- **小家系 informative marker 不足**：家系成员少时难以建立单体型追踪，需扩展测序 / 家族采血
- **无法检测的变异类型**：
  - 大片段复杂重排（位于重复区）
  - 某些线粒体异质性位点（需特殊 heteroplasmy 定量）
  - 新生突变未确认亲本来源

### 7.2 医学风险
- 促排卵药物：卵巢过度刺激综合征 (OHSS) 轻-重度
- 取卵操作：出血、感染（<1%）
- 多胎妊娠风险（中国已普遍单胚胎移植 eSET 降低该风险）
- 长远健康随访数据仍在积累

### 7.3 社会心理负担
- 周期失败挫败感、抑郁风险
- 费用与时间压力
- 家庭关系张力（尤其涉及致病变异来源亲属的愧疚）

## 8. 替代方案

### 8.1 自然妊娠 + 产前诊断
- **CVS（绒毛膜穿刺）**：11-13 周，可早期诊断，流产风险 ~0.5-1%
- **羊水穿刺**：16-20 周，流产风险 ~0.1-0.3%，标准方案
- **NIPT（无创产前检测）**：母血游离 DNA，主要用于常见三体 (T21/T18/T13) 和性染色体筛查；部分单基因病 (achondroplasia、thanatophoric dysplasia、某些新生显性) 有商用 cfDNA-based 检测，但适应证有限
- 发现受累 → 夫妇面临终止妊娠的艰难选择

### 8.2 其他生育方式
- **供精 / 供卵**：当一方为致病变异唯一来源时可选
- **胚胎领养**（中国目前法规不明朗，极少实施）
- **不生育 / 收养**

## 9. 与遗传咨询师配合

### 9.1 PGT-M 前必做
- 完整家系图（至少三代）
- 确诊家族致病变异（sanger 验证 / panel 阳性结果）
- ACMG 分级确认 P 或 LP（VUS 不进入 PGT）
- 夫妇双方基因咨询：风险、选项、技术局限、心理准备、伦理议题
- 对需追踪的 indel/大片段/重复区变异评估可检测性

### 9.2 多学科团队 (MDT)
- 临床遗传科（咨询 + 变异判读）
- 生殖医学科（IVF 周期管理）
- 产科（妊娠期监测 + 产前确认）
- 分子诊断实验室（PGT panel 设计 + 单体型分析）
- 伦理委员会
- 心理咨询（可选但建议）

## 10. 伦理与法律

### 10.1 中国法规
- 《人类辅助生殖技术管理办法》（2001 原国家卫生部发布、多次修订）
- 《人类辅助生殖技术规范》《人类精子库基本标准与技术规范》
- 《胚胎植入前遗传学诊断技术规范》

### 10.2 禁止条款
- **禁止非医学需要的性别选择**（X 连锁病例外，医学指征合理）
- **禁止"设计婴儿"**（非疾病性状选择如身高、外貌、智力）
- **禁止胚胎买卖**
- **禁止克隆用于生殖目的**
- **禁止生殖系基因编辑**（贺建奎事件后监管全面收紧）

### 10.3 患者知情同意要点
- 技术误判率（~1-2%）
- 未使用胚胎处置方案（冻存、销毁、科研捐赠）
- 隐私与数据保护
- 多周期可能性与经济负担
- 心理支持资源

## 11. 实务注意事项

### 11.1 准备阶段
- 材料清单：身份证、结婚证、病史记录、家系图、先证者分子检测报告
- 排队等待：部分机构 6-12 个月（节假日前后波动）
- 建议同时预约 1-2 家备选中心

### 11.2 基金会与项目资助
- 部分罕见病基金会（如瓷娃娃关爱中心、血友之家、SMA 之家）有专项资助或对接项目
- 地方政府罕见病项目（上海、广东、浙江等部分地区）
- 高校 / 科研中心临床试验可能免除部分费用（需符合入组标准）

### 11.3 海外 PGT 合规
- 中国公民赴海外（美、泰、俄等）做 PGT 合法性：
  - 医疗行为在境外合法不等于归国后一切合规
  - 非医学性别选择（海外合法地）归国后法律争议可能
  - 数据跨境、胚胎运输合规（中国禁止胚胎出境）
  - 建议事前咨询法律意见，避免灰色操作

### 11.4 常见患者困惑
- "检测后为什么还要产前诊断？"
  → PGT 有 1-2% 误判可能，产前是最终确认
- "为什么做了 PGT 还可能移植失败？"
  → IVF 本身活产率非 100%，胚胎染色体、子宫内膜、母体因素都影响
- "能不能用 PGT 选性别防止 XLR 发病？"
  → 在 XLR 明确家族致病变异情境下，选择不携带致病 X 的胚胎（含男女胚均可）符合医学指征；不得以"想要男/女孩"为由选择

### 11.5 咨询师输出示例
对于夹心饼干型家庭（一方 P 级携带 + 经济紧张 + 高龄）：
- 明示 PGT 代价与成功率
- 提供对比表：PGT-M vs 自然妊娠+产前
- 给出决策支持框架，但不替代决策
- 转介心理支持与病友社群
