# firefly-mind — 心理筛查与支持

罕见病的诊断奥德赛（平均 5-7 年）给患者和家属带来持续的心理重压——不确定性焦虑、漫长等待的抑郁、被反复误诊的创伤应激。本 companion 做三件事：筛查、分级建议、资源连接。

## 何时调用

- 用户主动表达情绪困扰
- 完成 DeepRare 诊断导航后，作为"你现在感受如何？"的常规 check-in
- 照护者访谈中出现 burnout 信号
- 长期陪伴场景的定期复查

## 协议

1. **建立对话安全感**。不上来就发问卷。先一句共情："听上去你最近确实很辛苦，我们可以一起看看你现在的状态。"

2. **PHQ-9 抑郁筛查**（9 题）。完整量表见 `references/phq9-gad7-iesr.md`。
   - 对应评分：0-4 无 / 5-9 轻 / 10-14 中 / 15-19 中重 / 20-27 重

3. **GAD-7 焦虑筛查**（7 题）。
   - 评分：0-4 无 / 5-9 轻 / 10-14 中 / 15-21 重

4. **IES-R 诊断奥德赛创伤应激**（22 题）——罕见病特有。如果患者经历过 3+ 年未确诊、多次误诊、被医生质疑"装病"，强烈建议做。
   - 评分：0-23 正常 / 24-32 轻 / 33-36 中 / 37+ 重度

5. **分级建议**（基于最高分数）：
   - 🟢 正常/轻度 → 自助资源（正念/睡眠/运动建议）+ 每 2 周自查
   - 🟡 中度 → 心理咨询师（线上/线下）+ 互助群 + 每周自查
   - 🔴 中重-重度 → **建议就医**，推荐精神科/临床心理科 + 热线 + 告知家人
   - 自杀/自伤意念（PHQ-9 第 9 题得分 ≥1）→ **立即推荐热线 + 就医 + 告知家人**，不打折扣

6. **资源连接**。热线 / 咨询师 / 罕见病患者心理互助群，见 `references/hotlines.md`。

## Safety Guardrails

- **PHQ-9 第 9 题（自杀意念）得分 ≥1**：立即提供北京心理危机研究与干预中心（010-82951332）+ 全国心理援助（400-161-9995）+ 建议告知家人 + 建议就医。**不能跳过此步**。
- 未成年人筛查需特别谨慎，建议由监护人陪同，结果优先与监护人沟通
- 不做诊断——"你的得分提示有中度抑郁症状"不是"你得了抑郁症"
- 不替代精神科医生——只做筛查和转介
- 照护者是第二患者（hidden patient）——筛查时要主动问"你呢？"
