# firefly-diet — 代谢性罕见病饮食管理

代谢性罕见病（尤其是先天性代谢缺陷 IEM）的饮食不是辅助治疗，而是核心治疗。错一顿饭可能代谢危象。

## 何时调用

- 患者已确诊代谢性罕见病（profile.json `confirmed_diagnosis` 在代谢病清单内）
- 家长问"这个病能吃什么"、"能喝普通奶粉吗"、"能吃水果吗"
- 医生开了特殊饮食但没给具体食物清单
- 患者处于特殊阶段（哺乳期、妊娠、生病发热、手术前后）——代谢危象高风险

## 协议

1. **识别病种**。从 profile.json 或用户输入提取确诊病种。对比 `references/metabolic-disease-diets.md` 病种清单。
2. **加载病种特异协议**：
   - **PKU**：苯丙氨酸 (Phe) 限制（婴儿 20-70 mg/kg/d，随年龄调整）；禁食高蛋白（肉蛋奶豆）；用专用氨基酸配方（PKU 特医奶粉）
   - **Pompe**：高蛋白饮食（30% 能量占比）+ 酶替代治疗辅助
   - **半乳糖血症**：严格禁乳糖（母乳/牛奶/奶酪/乳清蛋白/酸奶），用豆基/无乳糖配方
   - **尿素循环障碍**：限蛋白（0.8-1.2 g/kg/d）+ 高能量 + 必需氨基酸补充
   - **MSUD 枫糖尿症**：限支链氨基酸（Leu/Ile/Val）
   - 其他病种见 references
3. **食物交换份**：按照当前国内可获得的特医食品，给出每日食谱样板（早中晚三餐 + 加餐）。
4. **药食相互作用**：
   - ERT（酶替代治疗）与饮食时序
   - 抗癫痫药与蛋白摄入互动
   - 中药禁忌（某些代谢病绝对禁止温补/人参/当归）
5. **儿童生长发育监测**：身高体重 Z-score 每月、头围（<3 岁）、发育里程碑。低于 P3 或跌破轨道 → 紧急转诊代谢科营养师。
6. **代谢危象识别与应急**：呕吐/嗜睡/呼吸急促/行为改变 → 立即去急诊 + 告知代谢病诊断 + 禁食 + 静脉高糖。

## Safety Guardrails

- **本 companion 不替代临床营养师**。所有方案必须与患者主管营养师确认。
- 不给未确诊患者开饮食方案——先让 firefly-organize / firefly-genetic-counseling 确诊
- 特殊阶段（哺乳期、妊娠、发热、手术前后）必须转回营养师，不自行调整
- 中国上市的特医食品清单可能滞后，标注"该产品国内可获性需确认"
- 家长常见误区警示：「吃点蔬菜没事」—— 某些 IEM 蔬菜也含关键氨基酸
