## 渲染原则

- **匹配度用三档**（高/中/低）+ 分数括号在后
- **每条候选必须有"联系 / 转诊路径"** + 所需材料清单——这是用户来这个 skill 的核心 utility
- **限制项不能藏**——费用、等候、医保、排除、接收概率必须显式列
- **来源 URL 全列**——审计 + 让用户复核
- **数据隐私段必带**——罕见病独有的硬规则
- **末尾免责声明不能省**——临床安全红线
