# 中国罕见病资源库

## 罕见病诊疗协作网

全国罕见病诊疗协作网由国家卫健委牵头建立，覆盖 324 家医院。

### 国家级牵头医院
- 北京协和医院（全国罕见病诊疗牵头医院）
- 各省级牵头医院覆盖所有省份

### 按病种推荐科室的一般规则
| 系统 | 推荐科室 | 常见病种举例 |
|------|----------|-------------|
| 神经系统 | 神经内科 / 小儿神经内科 | 脊髓性肌萎缩症(SMA)、肌营养不良、线粒体病 |
| 代谢/内分泌 | 内分泌科 / 遗传代谢科 | 苯丙酮尿症(PKU)、戈谢病、法布雷病 |
| 血液系统 | 血液科 | 地中海贫血、血友病、阵发性睡眠性血红蛋白尿 |
| 免疫系统 | 风湿免疫科 / 免疫科 | 原发性免疫缺陷 |
| 骨骼系统 | 骨科 / 内分泌科 | 成骨不全症、软骨发育不全 |
| 皮肤 | 皮肤科 | 遗传性大疱性表皮松解症 |
| 眼科 | 眼科 | Leber 遗传性视神经病变、视网膜色素变性 |
| 心血管 | 心内科 | 肥厚型心肌病(遗传性) |
| 肾脏 | 肾内科 | Alport 综合征、常染色体显性多囊肾 |
| 多系统 | 医学遗传科 / 罕见病门诊 | 马凡综合征、结节性硬化症 |

### 推荐就诊策略
1. **首选**：当地省级协作网牵头医院的对应专科
2. **疑难**：北京协和医院罕见病门诊（多学科会诊，MDT）
3. **儿童**：北京儿童医院、上海复旦儿科、广州妇女儿童医中心
4. **基因检测解读**：选择有临床遗传学专家的医院

## 国家罕见病目录

### 第一批（2018年发布，121种）
涵盖 121 种罕见病，是医保保障和政策制定的基础。

### 后续更新
目录持续扩充中，关注国家卫健委官方发布。

### 目录的意义
- 目录内疾病：有明确的诊疗指南、部分有医保覆盖
- 目录外疾病：保障相对薄弱，但不代表没有帮助

## 患者组织

### 全国性组织
- **中国罕见病联盟（CHARD）**：全国性罕见病联合组织
  - 官网：提供罕见病信息、政策动态
  - 下设多个病种分组
- **病痛挑战基金会**：罕见病患者援助和倡导
  - 医疗援助项目
  - 罕见病就医地图
- **蔻德罕见病中心（CORD）**：信息服务和患者赋能

### 病种专属组织（部分示例）
| 疾病 | 组织 |
|------|------|
| 渐冻症(ALS) | 中国 ALS 协会、渐冻人关爱中心 |
| 瓷娃娃(成骨不全) | 瓷娃娃罕见病关爱中心 |
| 血友病 | 中国血友之家 |
| 白化病 | 月亮孩子之家 |
| 脊髓性肌萎缩症 | SMA 之家 |
| 马凡综合征 | 马凡之家 |
| 苯丙酮尿症 | PKU 之家 |

### 如何找到特定病种的患者组织
1. 中国罕见病联盟官网查询成员组织
2. 病痛挑战基金会罕见病地图
3. 微信搜索：疾病名 + "患者群" / "病友群"
4. 知乎/微博/小红书搜索患者经验分享

## 经济援助

### 国家层面
- 国家医保目录中的罕见病用药（已纳入约 50+ 种罕见病用药）
- 大病医疗救助（各地标准不同）
- 临时救助制度

### 地方层面
- 浙江、山东、上海等省市有罕见病专项保障政策
- "惠民保"等城市补充医疗保险覆盖部分罕见病用药
- 各地残疾人补贴政策

### 慈善援助
- 病痛挑战基金会医疗援助项目
- 各药企患者援助计划（PAP）
- 中华慈善总会罕见病项目
- 各地方慈善基金会

### 申请援助的一般步骤
1. 确认诊断（需要明确的诊断证明）
2. 确认所需药物/治疗
3. 咨询当地医保政策
4. 申请大病医疗救助
5. 联系患者组织了解额外援助渠道
6. 如需，申请药企 PAP 或慈善基金

## 遗传咨询资源

### 有资质的遗传咨询服务
- 各大三甲医院医学遗传科 / 产前诊断中心
- 中国遗传学会遗传咨询分会认证的遗传咨询师
- 在线遗传咨询平台

### 遗传咨询内容
- 遗传模式解读（对家庭成员的意义）
- 再生育风险评估
- 产前诊断选项：绒毛取样(CVS)、羊膜穿刺
- PGT（胚胎植入前遗传学检测）：适用于已知致病变异的家庭
- 携带者筛查
