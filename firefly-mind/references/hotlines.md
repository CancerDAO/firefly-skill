- **即刻危险**（已在实施自伤、有明确计划和工具）→ **立即拨打 120**，不要先拨心理热线

- **ICARE 患者组织能力建设项目**（病痛挑战基金会）：其中包含照护者心理赋能模块

### 线上心理咨询平台（筛选标准见第六节）

- **简单心理 JianDanXinLi**：平台化咨询，咨询师资质公开可查
- **壹心理**：大众心理平台，文章+咨询，咨询师筛选需自己判断
- **好心情**：三甲医院精神科医生为主，可在线问诊开药
- **春雨医生精神心理科**：线上问诊入口

> 注意：商业平台对咨询师资质审核程度不同，务必选择有国家二级/三级心理咨询师证、或精神科执业医师证、或临床心理学/精神医学硕博学位的咨询师。

## 六、安全声明与资源筛选原则

### 信息时效性

- 本清单所有号码、机构、项目核实于 **2026-04**
- 热线号码、运营时间、项目名可能发生变化，拨打前请通过官方渠道（机构官网/微信公众号/114）二次核实
- 发现失效条目可反馈至 firefly-skill 项目仓库：https://github.com/zwbao/firefly-skill/issues

### 网络咨询师 / APP 筛选原则

**只推三类**：

1. **政府运营**：12320 卫生热线、12355 共青团系统、各省市心理援助热线
2. **三甲医院互联网医院**：北医六院、安定、上海精卫、华西等官方互联网医院入口
3. **有明确资质证明的商业平台**：咨询师有国家二级/三级心理咨询师证、精神科执业医师证、临床心理学/精神医学硕博学位，且平台公开展示证件

**不推**：

- 无法验证资质的"情感导师""心理大师"
- 承诺"快速治愈""保证效果"的个人或机构
- 以卖课/卖高价课程为主要商业模式的"心理培训"类账号
- 未经注册/未公开资质的微信私人心理咨询师
- 任何声称能"治愈罕见病本身"的心理/灵修机构

### 紧急情况的兜底逻辑

```
即刻生命危险（已实施/即将实施自伤）：
  → 拨打 120 急救
  → 家人陪同到最近综合医院急诊
  → 不要先拨心理热线（不会派车）

强烈自杀意念、计划明确但未行动：
  → 拨打心理危机热线（010-82951332 或 400-161-9995）
  → 同时联系家人/信任的朋友陪同
  → 24-72 小时内到精神科 / 急诊心理科就医

情绪困扰、睡眠问题、无自杀意念：
  → 心理援助热线 或 心理咨询
  → 按 firefly-mind 分级建议执行
```

### 最后提醒

罕见病患者和家属长期处于应激状态，"扛住了"不等于"没事"。任何时候觉得"撑不下去了"，热线是免费的，不会评判你，请先打。
