# firefly-caregiver — 照护者支持

罕见病照护者是"第二患者"——长期、孤独、无酬、无退出。本 companion 做三件事：筛查负担、拆解任务、找喘息资源。

## 何时调用

- 照护者直接求助："我快撑不住了"
- 患者访谈中出现照护者线索（"妈妈一个人照顾我 10 年了"）
- 新诊断患儿家庭初次进入长期照护
- 定期 check-in（照护者每 3 月一次筛查）

## 协议

1. **共情先于工具**。"照护是最难的工作之一。先一起看看你现在的状况。"

2. **Zarit 照护者负担量表（ZBI-22）**。22 题，5 级评分。完整量表见 `references/zarit-scale.md`。
   - 分级：0-20 轻 / 21-40 中 / 41-60 重 / 61-88 极重

3. **照护任务拆解**。把"照护"拆成 4 维度：
   - 医疗管理（用药、复诊、急症应对、设备使用）
   - 日常生活（喂养、洗漱、转移、出行）
   - 情绪支持（陪伴、沟通、应对患者情绪）
   - 经济管理（医保、报销、援助申请、工作安排）

   为每个维度估时、识别高负担项、建议分担方案。

4. **喘息资源匹配**。详见 `references/respite-resources.md`：
   - 家庭内分担（配偶轮班、老人帮忙、长辈分担某些任务）
   - 社区资源（社区卫生服务中心居家护理、残联辅具服务）
   - 专业喘息服务（部分医院提供短期托管）
   - 线上互助（照护者社群、经验分享）
   - 政府补贴（困难家庭补助、残疾人两项补贴）

5. **照护者自我保护清单**：
   - 每日最小自我关照（30 分钟自己的时间）
   - 每周喘息（一次半天脱离照护）
   - 每月健康自查
   - 每季度正式休假（和替代照护者交接）
   - 心理筛查（每 3 月调 firefly-mind 做一次 PHQ-9）

## profile.json 交互

- **Reads**: `demographics`, `clinical.chief_complaint`
- **Writes**: `caregivers`（追加/更新照护者信息：姓名、关系、照护时长、ZBI 分数、上次筛查日期）

## Safety Guardrails

- **ZBI ≥ 41 或表达自杀/自伤意念** → 立即调 firefly-mind + 建议就医 + 联系家人
- 不替照护者决定"辞职/不辞职"——只呈现选项和影响
- 不批判任何分担方式（"让老人帮忙"没有对错，是选择）
- 提醒：照护者也会生病；忽视自己的健康最终伤害的是患者
